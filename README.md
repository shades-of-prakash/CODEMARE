## About the Project

CodeMare is a full-stack web application built with Next.js, designed to bring coders together to solve problems collaboratively. Users can post coding challenges—complete with code snippets—and others can respond with solutions or suggestions in the comments. It’s a dynamic, community-driven platform for learning and problem-solving.

---

## Features
- **Problem Posting**: Share coding issues with a title, description, and formatted code snippet.
- **Comment-Based Solutions**: Community members can reply with answers or feedback directly in the comments.
- **Server-Side Rendering (SSR)**: Fast page loads and SEO-friendly content with Next.js SSR.
- **API Routes**: Built-in backend API for handling posts, comments, and user data.
- **Syntax Highlighting**: Code snippets rendered with proper formatting (Prism.js).
- **Responsive Design**: Works seamlessly on desktop and mobile devices.
- **Authentication**: Sign in to post or comment ( NextAuth.js ).
- **Search & Tags**: Filter problems by keywords or categories.

---

## How It Works
1. **Post a Problem**: Create a new post with your coding challenge.
2. **Community Solves**: Browse posts and leave solutions or insights in the comments.
3. **Collaborate**: Engage in discussions to refine solutions and learn from each other.

## Tech Stack
- **[Next.js](https://nextjs.org/)** – React framework for full-stack development (frontend + backend).
- **[React](https://reactjs.org/)** – UI library for building interactive components.
- **[Node.js](https://nodejs.org/)** – Runtime for server-side logic.
- **[MongoDB](https://www.mongodb.com/)** – Database for storing posts and comments (PostgreSQL).
- **[Vercel](https://vercel.com/)** – Deployment platform.
- **[NextAuth.js](https://next-auth.js.org/)** – Authentication for user management.
- **[Prism.js](https://prismjs.com/)** – Syntax highlighting for code snippets
